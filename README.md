🌞 SunCart – Summer Essentials Store
📌 Description

SunCart is a summer-themed eCommerce web app where users can browse and purchase seasonal products like sunglasses, outfits, skincare, and accessories. It includes authentication, protected routes, and a fully responsive design.

🚀 Live Link

👉 https://sun-cart-b13-a8-bv7w.vercel.app/

💻 GitHub Repo

👉 https://github.com/fariaesha-prog/Sun-Cart-B13-A8

✨ Features
User authentication (Email/Password + Google)
Protected product details page
Product listing from JSON data
My Profile & update user info
Responsive design (mobile, tablet, desktop)
Summer-themed UI with extra sections
🛠️ Tech Stack
Next.js
Tailwind CSS + DaisyUI / HeroUI
BetterAuth
